# Recall - finesse brief

register: product
one-liner: 从本机 Codex 与 Claude Code 会话提炼可审查 Meta 和 Skill 的 macOS 工具

## Users & context
在 macOS 上同时使用 Codex 与 Claude Code 的开发者，需要在本机检查、批准和复用 Agent 工作经验

## Brand personality
soul: 克制、可信、可审计
voice: 直接、清晰、技术性

## Locked dials
SOUL: 6   SPECTACLE: 2   DENSITY: 8

## Visual lock
accent: forest green #2f6b4f, with amber reserved for review state
theme: light content with a dark rail, suited to a focused desktop workbench
type: macOS system sans for interface clarity, mono only for paths and protocol data

## Anti-references
- first-order: AI 工具常见的紫蓝霓虹和渐变按钮
- second-order: 纯黑终端界面和荧光绿开发者工具
- 不做营销落地页，不使用巨型 Hero、玻璃卡片或装饰性动效

## Notes
所有数字来自真实本地状态；原始会话只读；视觉反馈只表达扫描、反思、审核和连接状态
